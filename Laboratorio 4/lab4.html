Preguntas a Responder

  • ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

    • Java y JavaScript son dos lenguajes de programación diferentes con diferentes usos. Java es un lenguaje de programación orientado a objetos utilizado para desarrollar aplicaciones de escritorio y aplicaciones móviles, mientras que JavaScript es un lenguaje de programación interpretado utilizado principalmente para desarrollar aplicaciones web.

    Algunas de las diferencias clave son:

    • Java es un lenguaje de programación compilado, mientras que JavaScript es interpretado.
    • Java se ejecuta en una máquina virtual, mientras que JavaScript se ejecuta en un navegador web.
    • Java es un lenguaje de programación orientado a objetos fuertemente tipado, mientras que JavaScript es un lenguaje de programación débilmente tipado y orientado a objetos.

    Algunas de las similitudes son:
    • Ambos lenguajes de programación utilizan sintaxis similar a la de C.
    • Ambos lenguajes de programación son orientados a objetos.
    • Ambos lenguajes de programación tienen una gran cantidad de bibliotecas y herramientas disponibles para su uso.

  • ¿Qué métodos tiene el objeto Date?
    1. getDate(): Devuelve el día del mes (1-31) para la fecha especificada.
    2. getMonth(): Devuelve el mes (0-11) para la fecha especificada.
    3. getFullYear(): Devuelve el año (cuatro dígitos) para la fecha especificada.
    4. getHours(): Devuelve las horas (0-23) para la fecha especificada.
    5. getTime(): Devuelve el número de milisegundos desde el 1 de enero de 1970 hasta la fecha especificada.

  • ¿Qué métodos tienen los arreglos?
    1. push(): Agrega un elemento al final del arreglo.
    2. pop(): Elimina el último elemento del arreglo.
    3. shift(): Elimina el primer elemento del arreglo.
    4. unshift(): Agrega un elemento al inicio del arreglo.
    5. splice(): Agrega o elimina elementos en el arreglo.

  • ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    • El uso de variables globales dentro de funciones puede tener varias implicaciones, algunas de las cuales incluyen:

    • Puede hacer que el código sea más difícil de leer y entender, especialmente si se utilizan muchas variables globales.
    • Las variables globales pueden ser sobrescritas accidentalmente por otras partes del código.
    • El uso excesivo de variables globales puede hacer que el código sea más difícil de depurar.
    • Las variables globales pueden hacer que el código sea más difícil de reutilizar en otras partes del programa.
